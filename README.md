# bioskop_rumah

Bioskop Rumah adalah aplikasi berbasis flutter yang dapat menampilkan daftar dari film populer
yang di dalamnya terdapat:
1. Fitur search film
2. Rating film 
3. Sinopsis film
4. Tanggal rilis film

Daftar film dari aplikasi ini diambil dari API TMDb
